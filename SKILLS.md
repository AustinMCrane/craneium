# Your Skills

Your skills can be found in the `./skills` directory. Each subdirectory contains a `SKILL.md` with documentation for that skill. **Always scan `./skills/` at the start of a session** to discover what skills are available — do not assume a fixed list.

Each skill will have a specific purpose and function you can use to assist your person in their work.

## Skill Storage

Store new skills in the `./skills` directory as a new subdirectory with a `SKILL.md` file inside. The format should be markdown.
